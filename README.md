Introduction
============

webgpu-ts is an open source for implementing basic examples such as 3D Rendering, Image Processing, Light Effect, etc on webgpu-ts. It started to study and easily apply to a blog or web site.


Requirements
============
- Node >= 12.18.3
- NPM >= 6.14.6
- YARN >= 1.22.5
